# eVoting-System

Online voting system is an online voting
technique. In this system people who are
authorized by the admin can cast his/her
vote online without going to any physical
polling station. There are many voting
procedures which are being used for Voting
purpose, such as ballot paper, EVM
machine but all these procedures require
more time and more man power so to
eliminate all these drawbacks we provide
an online voting system which provides
features such as accuracy, convenience,
flexibility, privacy and verifiability.
